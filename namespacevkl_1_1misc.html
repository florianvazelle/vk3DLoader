uint32_t vkl::misc::findMemoryType(VkPhysicalDevice physicalDevice, uint32_t typeFilter, VkMemoryPropertyFlags properties)
#include <common/misc/Device.hpp>

Get the index of a memory type that has all the requested property bits set.

Parameters
physicalDevice
typeFilter Indique les types de mémoire que l'on veut trouver (chaque n-ième correspond a un type de mémoire)
properties Bitmask spécifiant les propriétés d'un type de mémoire
Returns Indice de la mémoire
Exceptions
Throws an exception if no memory type could be found that supports the requested properties
